## [1.0.0] - 2025-11-29

### 新增
- ✨ 一键安装 Alist 功能
  - 自动检测系统架构（AMD64/ARM64/ARMv7）
  - 从 GitHub 获取最新版本
  - 自动安装系统依赖
  - 创建 systemd 服务
  - 自动启动并显示管理员密码

- ✨ 完整卸载功能
  - 停止并禁用服务
  - 清理所有文件和配置
  - 双重确认防止误操作

- ✨ 服务管理功能
  - 启动 Alist 服务
  - 停止 Alist 服务
  - 重启 Alist 服务
  - systemd 集成管理

- ✨ 状态查看功能
  - 显示服务运行状态
  - 显示版本信息
  - 显示监听端口
  - 显示访问地址
  - 显示安装路径
  - 显示 systemd 服务详情

- ✨ 端口修改功能
  - 显示当前端口
  - 验证端口范围（1-65535）
  - 检测端口占用
  - 自动更新配置
  - 重启服务应用更改

- ✨ 密码管理功能
  - 重置管理员密码
  - 生成随机强密码
  - 实时显示新密码

- ✨ 日志查看功能
  - 显示最近 50 行日志

- ✨ 在线更新功能
  - 检查最新版本

  - 下载安装新版本
  - 失败自动回滚

- ✨ 用户界面
  - 彩色交互式菜单
  - 清晰的提示信息
  - 友好的错误处理

## 版本说明

### 版本命名规则
- 主版本号：重大架构变更或不兼容的 API 修改
- 次版本号：向下兼容的功能性新增
- 修订号：向下兼容的问题修正

### 升级建议
- 从旧版本升级前，建议备份 `/opt/alist/data` 目录
- 重大版本升级请先在测试环境验证
- 查看本更新日志了解具体变更内容
